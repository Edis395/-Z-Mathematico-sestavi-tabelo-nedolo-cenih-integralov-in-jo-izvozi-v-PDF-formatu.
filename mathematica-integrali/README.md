# Tabela nedoločenih integralov z Mathematico

## Povzetek projektne naloge

V tej projektni nalogi s pomočjo programskega orodja Wolfram Mathematica sestavimo strukturirano tabelo izbranih nedoločenih integralov. Tabela zajema osnovne integrale elementarnih funkcij (polinomske, trigonometrične, eksponentne in logaritemske funkcije) ter prikazuje tako integrand kot analitični rezultat integracije. Končno tabelo izvozimo v format PDF, ki je primeren za tisk in deljenje.
